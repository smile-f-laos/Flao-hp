# Smile F LAOS HP運用マニュアル

このマニュアルは、Smile F LAOSのホームページを後輩が安全に更新・管理できるようにするためのものです。

## 1. このHPの目的

このHPは、Smile F LAOSの活動内容、ラオスコーヒー、スタディツアー、協力団体、お知らせを外部に伝えるための公式ホームページです。

## 2. 基本の更新手順

1. GitHubにログインする
2. エフラオHPのリポジトリを開く
3. 編集したいHTMLファイルを開く
4. 鉛筆マークから編集する
5. 内容を変更する
6. Previewで確認する
7. commit messageを書く
8. Commit changesを押す
9. 公開されたHPを確認する

## 3. よく編集するファイル

| ファイル名 | 内容 |
|---|---|
| index.html | トップページ |
| coffee.html | コーヒーページ |
| study-tour.html | スタディツアーページ |
| partners.html | 協力団体ページ |
| news.html | お知らせページ |

## 4. 更新するときの注意

- 文章を変更したら、誤字脱字を確認する
- 団体名・企業名・人名を間違えない
- 画像ファイル名を変更した場合は、HTML側の記述も直す
- スマホ表示でも崩れていないか確認する
- 大きな変更をするときは、代表や広報担当に確認する

## 5. commit messageの例

- トップページの文章を修正
- コーヒーページの商品説明を更新
- お知らせページにイベント情報を追加
- スタディツアーページの写真を差し替え

## 6. 更新後チェックリスト

- [ ] 編集したファイルは正しいか
- [ ] 誤字脱字はないか
- [ ] 日付・場所・時間は正しいか
- [ ] 画像は表示されているか
- [ ] リンク切れはないか
- [ ] PC表示で崩れていないか
- [ ] スマホ表示で崩れていないか
- [ ] 公開後のHPを確認したか
